# om-tutorial

A simple Om.Next tutorial app using Datascript and Boot

## Usage

in a terminal:
```
boot dev
```

in another terminal:
```
boot repl -c
(start-repl)
(js/console.log "hello from the CLJS repl!")
```

to update a project.clj for integrating with IntelliJ:
```
boot lein-generate
```
